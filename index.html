Welcome to artEngine!

artEngine is a JavaScript library for creating amazing vector-based graphics; created with simplicity and ease of use in mind. Create beautiful images with artEngine and the HTML5 canvas with extremely minimal code. Turn 100+ lines of code into a mere few! Check out the examples to see some basic implementations of the library.

With artEngine, anything is possible. Anything from a professional logo to the header on your friend's blog, an image for a 2D tower-defense game to a picture for your living room wall, and everything in between. What will you build next?
